اگزوتیک با مهدیار — ورود با توکن گیت‌هاب

 
چطور توکن بسازم؟ (یک‌بار)
۱. برو به github.com/settings/personal-access-tokens/new
۲. یک نام بذار (مثلاً exotics-admin) و مدت انقضا را طولانی بگذار.
۳. در Repository access → Only select repositories → مخزن exoticsbymahdiyar را انتخاب کن.
۴. در Permissions → Repository permissions → Contents را روی Read and write بگذار.
۵. Generate token را بزن، توکن را کپی کن و اینجا بچسبان.
توکن فقط در همین مرورگر ذخیره می‌شود، نه جایی دیگر.
